# MobileSAM Prompt Pilot Findings

## Objective

The goal of this pilot was to verify whether MobileSAM can generate useful
candidate lesion masks for the ISIC-Fitzpatrick segmentation pilot using a
simple, reproducible prompt strategy. This step is intended for visual
inspection and method validation before any larger batch processing.

## Configuration

The pilot used MobileSAM with the `vit_t` model and the local checkpoint:

`models/mobile_sam/mobile_sam.pt`

Inputs were the 1024x768 resized inference images from the segmentation pilot.
For each image, the prompt consisted of one positive point at the image center
and four negative points at the image corners. Inference used
`multimask_output=True`, producing three candidate masks per image.

## Technical Result

The model runs locally on CPU and generates three candidate masks per image.
The pilot script writes each binary mask and a red semi-transparent overlay for
visual review.

## Methodological Result

The visual quality of the generated masks is not reliable enough to use
automatically. The outputs should be treated as candidate masks for manual
review, not as final segmentation labels.

## Observed Failure Modes

Preliminary visual review showed several important failure modes:

- Low contrast between lesion and surrounding skin.
- Non-circular, irregular, or dispersed lesions.
- Multiple lesions in the same image.
- Artifacts or tonal changes introduced by the transparent dermatoscope tip.
- Selection of surrounding skin or background instead of the lesion.

These observations are qualitative and come from pilot inspection. They should
not be interpreted as formal performance estimates.

## Decision

MobileSAM should not be used as an automatic generator of final masks in this
project. It can still be useful as a generator of candidate masks for manual
review, or as a limited baseline that documents the weaknesses of prompt-based
zero-shot segmentation in this acquisition setting.

Masks generated by this pilot should not be used for downstream classification
unless they pass an explicit manual quality-control step.

## Relevance to the Thesis

This pilot supports the need for an explicit mask validation stage before
training or evaluating classifiers with segmented images. Segmentation itself
can introduce bias if failure rates differ with lesion-skin contrast, skin tone,
or acquisition artifacts. For that reason, segmentation quality should be
documented and reviewed rather than assumed from model output alone.
